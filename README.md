# Odyssey_1

Перед тем, как космический корабль отправится бороздить просторы Вселенной, необходимо пригласить на борт экипаж, который будет состоять из 2 людей, 1 собаки и 1 кота.
В методе createCrew() добавь необходимое количество экземпляров соответствующих классов в список astronauts.

Подсказка:
Чтобы добавить объекты разных классов в один список, им нужен общий предок. Унаследуй интересующие тебя классы от Astronaut. Кто угодно может стать исследователем космоса :)

Псевдокод:

```
// Класс Astronaut (базовый класс)
КЛАСС Astronaut:
    МЕТОД getInfo() ВОЗВРАЩАЕТ СТРОКА:
        ВОЗВРАТ "Неизвестный астронавт"

// Класс Human наследуется от Astronaut
КЛАСС Human НАСЛЕДУЕТ Astronaut:
    МЕТОД getInfo() ВОЗВРАЩАЕТ СТРОКА:
        ВОЗВРАТ "Человек"

// Класс Dog наследуется от Astronaut
КЛАСС Dog НАСЛЕДУЕТ Astronaut:
    МЕТОД getInfo() ВОЗВРАЩАЕТ СТРОКА:
        ВОЗВРАТ "Собака"

// Класс Cat наследуется от Astronaut
КЛАСС Cat НАСЛЕДУЕТ Astronaut:
    МЕТОД getInfo() ВОЗВРАЩАЕТ СТРОКА:
        ВОЗВРАТ "Кот"

// Основной класс программы
КЛАСС Solution:
    // Список для хранения астронавтов
    ПЕРЕМЕННАЯ astronauts = новый список Astronaut

    // Главный метод программы
    МЕТОД main(args):
        ВЫЗВАТЬ createCrew()
        ВЫЗВАТЬ printCrewInfo()

    // Метод создания экипажа
    МЕТОД createCrew():
        ДОБАВИТЬ В astronauts новый Human()
        ДОБАВИТЬ В astronauts новый Human()
        ДОБАВИТЬ В astronauts новый Dog()
        ДОБАВИТЬ В astronauts новый Cat()

    // Метод вывода информации об экипаже
    МЕТОД printCrewInfo():
        ВЫВЕСТИ "На борт погружены члены экипажа: "
        ДЛЯ КАЖДОГО astronaut В astronauts:
            ВЫВЕСТИ astronaut.getInfo()
```

Краткое описание логики:

1. Есть базовый класс Astronaut с методом getInfo()
2. Три класса-наследника (Human, Dog, Cat) переопределяют метод getInfo() для возврата своих названий
3. В основном классе Solution создается список астронавтов
4. В список добавляются два человека, одна собака и один кот
5. Выводится информация о каждом члене экипажа с использованием полиморфизма (каждый объект вызывает свою версию getInfo())

Ожидаемый вывод программы:

```
На борт погружены члены экипажа: 
Человек
Человек
Собака
Кот
```
